# Source Analysis Documents

Status: historical/reference area by default
Scope: DOC-001C documentation navigation

## Important

Documents in this directory are historical or reference material by default.

They include source evaluations, connector candidate notes, exploratory analysis,
market-sensor findings, employer-origin experiments, and historical build
context.

They are not the primary place to learn the current architecture.

## Current system entry points

Use these instead for the current system:

- `docs/README.md`
- `docs/architecture/current_system_overview.md`
- `docs/architecture/system_diagrams.md`
- `docs/governance/README.md`
- `docs/governance/agent_governance_registry.md`

## When source-analysis docs are still useful

They are useful for:

- tracing why a source was explored,
- understanding historical source decisions,
- reviewing old connector candidates,
- recovering detailed source-specific context,
- validating that old lessons were not forgotten.

## What they are not

They are not automatically:

- current source strategy,
- current connector strategy,
- current architecture,
- current governance,
- current dashboard/product definition.

## DOC-001 policy

DOC-001 may later build a source-analysis archive index and deprecate or move
older files. Until then, treat this directory as historical/reference unless a
file explicitly states otherwise.
